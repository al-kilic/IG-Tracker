# IG Tracker

> A privacy-first Instagram follower analysis tool.

**Under construction.** Not ready for use yet.

Upload the ZIP file from Instagram's "Download Your Information" feature — see who unfollowed you, who doesn't follow back, and track changes over time.

**Your data never leaves your browser.** No password required. No server uploads. 100% TOS-compliant.
